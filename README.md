# e-plantShopping

e-plantShopping is a React-based shopping cart application developed as part of a Coursera final project.

## Project Overview
The Paradise Nursery shopping cart application allows users to browse a variety of house plants, add them to a shopping cart, update quantities, and view the total cost dynamically.

## Features
- Landing page with navigation
- Product listing with plant details
- Add to cart functionality
- Shopping cart with quantity controls
- Responsive UI built with React

## Technologies Used
- React
- Vite
- JavaScript
- CSS
- Redux Toolkit

This repository contains the complete source code for the e-plantShopping application.
